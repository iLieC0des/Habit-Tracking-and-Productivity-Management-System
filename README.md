Habit Tracking and Productivity Management System

A modern full-stack productivity and habit management application designed to help users build consistency, track progress, and improve daily productivity through habit analytics and streak-based tracking.

Overview

This project was developed as part of my MCA major project and focuses on creating an intuitive and scalable productivity platform. The application allows users to manage daily habits, maintain streaks, monitor performance, and stay consistent with long-term goals.

The system combines backend development, database management, and responsive frontend design to deliver a smooth and efficient user experience.

---

Features

- Secure user authentication system
- Create, update, and manage habits
- Daily habit check-in system
- Streak tracking and progress monitoring
- Productivity analytics dashboard
- Responsive and user-friendly interface
- Dynamic rendering using Flask and Jinja templates
